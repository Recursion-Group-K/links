|                                         | 用途|
|-----------------------|-----------------------|
|git clone {url}                          |	リモートからローカルにコピー |

|git commit 	                            |
|git push	                                |
|git switch	                              | ブランチ切り替え |
|git switch -c	                          | ブランチ作成&切り替え |

|git pull origin develop --rebase	develop | ブランチを最新の状態にする |
